# Security Policy

## Reporting a Vulnerability

If you discover a security issue in CoherenceBench, please report it
responsibly by emailing **research@pranaalpha.com** instead of using the
public issue tracker.

Please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to resolve confirmed
issues within 30 days.

## Scope

CoherenceBench is a research framework that interacts with external LLM APIs.
Security concerns may include:

- API key exposure in logs or output files
- Unsafe handling of model responses
- Dependency vulnerabilities
